Timezone Wrapper Element

Element which executes other elements in the selected timezone mode which can be Default, Client and Set.

Parameters:

  • Context ID - ID of the referenced IAppointmentsContext
  • Result Variable Name - Default status, name of the variable that contains additional information ( timezone object of the selected timezone). In order to access the name of the timezone do it like this: ${status.timezone.getName()}

Flows:

  • Elements - elements to be executed in the selected timezone mode.
